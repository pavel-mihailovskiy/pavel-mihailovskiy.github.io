Most of my career has been building and operating web products on Ruby on Rails — services, REST APIs (including Grape), and async work with Sidekiq. I design data models and tune queries across PostgreSQL, MySQL, and MongoDB, and use ElasticSearch / Redis when search or caching is part of the product.

On the frontend I ship UIs in JavaScript and TypeScript with React, and I’ve also delivered with Vue, Stimulus, and Node.js services when that fit the stack. Recent work includes Go services, LLM-backed workflows, and deep CRM/ATS integrations.

Quality is non-negotiable: RSpec (and Capybara where UI coverage matters), code review, and CI/CD so changes land safely. I join existing teams as a hands-on senior — contributing alongside others and helping the group ship with more confidence and consistency.
